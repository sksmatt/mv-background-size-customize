== Description ==

This plugin for WordPress adds an option to modify the body background-size property trough the new theme customize controls ( WordPress 3.4+ ). To work, the theme must support the built-in custom background feature.


== Installation ==

1. Unzip files.
2. Upload the mv-background-size-customize folder into your plugins directory.
3. Activate the plugin.
4. Have fun!
